# Crafty Publisher PRD

Build a single-purpose web tool that turns a WPRM recipe-card JSON export plus
edited food photos into WordPress-ready outputs.

The app must support file upload, image processing, vision-model-assisted alt
text drafting, dynamic forms, generated WordPress block markup, copy to
clipboard, downloadable text files, and metadata embedding.

The customer publishes to a recipe blog, but this product is not itself a blog
or static content site. It is a browser-based workflow tool.
